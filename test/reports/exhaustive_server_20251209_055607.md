## Key

### Modes
- **S (Short)**: 0.6 second interleave - lower latency, less robust
- **L (Long)**: 4.8 second interleave - higher latency, more robust
- Data rates: 75, 150, 300, 600, 1200, 2400 BPS

### Channel Conditions
- **Clean**: No impairments (SNR=30dB, no offset, no multipath)
- **AWGN_xxdB**: Additive White Gaussian Noise at specified SNR
- **Freq_xxHz**: Frequency offset in Hz
- **MP_xxms**: Multipath with specified delay spread
- **Combo**: Combined impairments (realistic HF conditions)

### Metrics
- **Pass Rate**: Percentage of tests that decoded correctly
- **BER**: Bit Error Rate (0.0 = perfect, higher = more errors)
- **Rating**: A+ (100%), A (95%+), B (80%+), C (60%+), F (<60%)
